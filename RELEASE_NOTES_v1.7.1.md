# Release v1.7.1: Premium Cursor Arrow & Custom Clearing Policy UI

This release introduces visual and interaction polishes to both the Remote Control interface and the application's configuration screens.

### Key Changes

* **Classic Cursor Arrow**:
  * Replaced the violet pointer dot (`●`) with a high-visibility classic cursor arrow (white filled with a crisp black outline).
  * Aligned the cursor's hot spot (the arrow tip) exactly with target touch coordinates.
  * Anchored click scaling animations to pivot directly from the arrow tip.
* **Higher Refresh Rate**:
  * Halved touch movement accumulation delay from `1000ms` to `500ms` for smoother, more responsive remote trackpad tracking.
* **Custom Selection Popups**:
  * Replaced native system dialog alerts for **Session Clearing Policy** and **Inactivity Duration** with premium custom-designed dark-themed overlays that perfectly match the app's overall midnight aesthetic.
